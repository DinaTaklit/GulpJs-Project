# Gulp Js Project

This project aim to learn Gulp js

## Create A Professional Structure And Your First Task

- Import gulp from gulp modules
  
    ```js
    const gulp = require('gulp');
    ```

- declare task using
  
    ```js
    gulp.task("taskname",funtion(){
        // Your task here
    });
    ```

- To run a task you run this command

    ```bash
        gulp task-name
    ```
